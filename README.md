# Pochi 🌿

> Keep your Paybill. Keep your WhatsApp group. We make the bookkeeping automatic.

Pochi is a WhatsApp-native fundraising coordination platform for Kenya. It connects to organizers' existing M-Pesa Paybills and Tills, automatically matches incoming payments to contributors, and posts live updates to WhatsApp groups.

## Quick Start

1. Read `CLAUDE.md` for the full project briefing
2. Read `ARCHITECTURE.md` for the technical blueprint
3. See `ui-designs/` for all page designs
4. Follow the build sequence in `CLAUDE.md` Section "Build Sequence"

## Tech Stack
- Backend: Node.js + Express + PostgreSQL + Prisma
- Frontend: React + Vite + TailwindCSS
- M-Pesa: Safaricom Daraja API
- WhatsApp: Africa's Talking Business API
